This is the project repository of Eukaryode, an evo-sim game being developed by Tom Slade.

To launch Eukaryode, open Demo/Eukaryode.exe.
WASD is camera movement. QE is camera rotation. +- is camera zoom.
'P' interpolates the tectonic map (press repeatedly to demonstrate tectonic interpolation).

To view Eukaryode's source code, open Eukaryode.sln.
Eukaryode/Model/Eukaryode.cs is the game's entry-point.
Amino is a custom library supporting the Eukaryode game.
